ReadMe

Runs in a console
java Console.txt

At any time, enter ‘quit’ to exit the program

You will always be prompted with one of two choices:
-Entering a file:
    Please enter a file name...
    Please enter the data…
 => Type the desired string and hit enter

-Entering a choice:
    What would you like to do?
    (1) Add students
    (2) Add assignments
 => Type a (valid) number and hit enter

Changes are not saved until you manually output them to a file!
